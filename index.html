💾 Python Installation Guide
Option A: Install Python from python.org (Recommended for General Use)
1Go to https://www.python.org/downloads/ and click the big yellow "Download Python 3.x" button.
2Run the installer. ⚠️ CRITICAL: Check the box that says "Add python.exe to PATH" at the bottom of the first screen. This is the #1 mistake people make!
3Click "Install Now" (the default). It installs to your user folder.
4When done, click "Disable path length limit" if it shows up, then Close.
5Open Command Prompt (press Win+R → type cmd → Enter) and type: python --version — you should see the version number.
Option B: Install Anaconda/Miniconda (Recommended for Data Science & GIS)
1Go to https://www.anaconda.com/download (Anaconda — big, includes everything) or https://docs.conda.io/en/latest/miniconda.html (Miniconda — minimal).
2Download the Windows 64-bit installer and run it.
3Accept defaults. Check "Add to PATH" if given the option (Anaconda may warn against it, but it's usually fine).
4Open Anaconda Prompt from the Start Menu and type: conda --version
5Create environments: conda create -n myproject python=3.11 then conda activate myproject
Option C: Using Python in ArcGIS Pro
1ArcGIS Pro comes with its own Python (via Conda). Open ArcGIS Pro → Project tab → Python.
2The default environment is arcgispro-py3. You can clone it to install additional packages without breaking the default.
3To open a Python console inside ArcGIS Pro: go to Analysis tab → Python → choose Python Window or Python Notebook.
4To run .py scripts outside ArcGIS Pro, open Python Command Prompt from Start Menu (installed with ArcGIS) and run: python my_script.py
5Install extra packages in cloned environment: conda install -n myenv pandas matplotlib or pip install package_name
How to Actually Open & Run Python
🖥️ 5 Ways to Open Python
1Interactive Shell: Open Command Prompt or PowerShell, type python and press Enter. You'll see >>>. Type code line by line. Type exit() to leave.
2Run a .py Script: Save your code in a file like myscript.py. In Command Prompt, navigate to the folder (cd C:\myfiles) and type python myscript.py
3IDLE: Search "IDLE" in Start Menu. It's a simple editor that comes with Python. Write code, press F5 to run.
4VS Code: Download VS Code from code.visualstudio.com. Install the Python extension. Open a .py file → click the ▶️ play button.
5Jupyter Notebook: Run pip install notebook then jupyter notebook in cmd. A browser window opens where you write code in cells.
Troubleshooting Common Install Problems
"python is not recognized" — Python is not on your PATH. Reinstall and check "Add to PATH", or manually add C:\Users\YourName\AppData\Local\Programs\Python\Python3x\ to your system PATH.
"pip is not recognized" — Try python -m pip install ... instead, or add Python's Scripts folder to PATH.
Multiple Python versions conflict — Use py -3.11 to pick a specific version (Windows Python Launcher). Use virtual environments!
"Permission denied" — Run cmd as Administrator, or use pip install --user package_name.
ArcGIS Pro package conflicts — Always clone the default environment before installing packages: Project → Python → Manage Environments → Clone.
Virtual Environments (Keep Projects Clean)
Create isolated Python environments to avoid package conflicts:
# Create a virtual environment
python -m venv myenv

# Activate it (Windows)
myenv\Scripts\activate

# Activate it (Mac/Linux)
source myenv/bin/activate

# You'll see (myenv) in your prompt when active

# Install packages (only for this project)
pip install numpy pandas matplotlib

# Show installed packages
pip list

# Deactivate when done
deactivate

# Delete environment (just delete the folder)
rmdir /s myenv  # Windows
rm -rf myenv    # Mac/Linux
